Análisis de Ventas de Vehículos Volkswagen

Autor: Ignacio Larreteguy

Este repositorio contiene la documentación inicial y el dataset utilizados
para el análisis del desempeño comercial y financiero de los modelos Volkswagen
en el período 2021–2025.

El objetivo del proyecto es evaluar la conveniencia de discontinuar modelos
con bajo desempeño, considerando volumen de ventas y rentabilidad.
